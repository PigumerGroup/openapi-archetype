Usage
=====

```
$ java -jar openapi-generator-util-0.1.0-all.jar download init generate
$ mvn -DaltDeploymentRepository=<Repository Id>::default::<Repository Url> clean deploy 
```

```
$ cd example
$ export SERVER_SERVLET_CONTEXT_PATH=/example
$ mvn spring-boot:run
```

or

```
$ cd example
$ export SERVER_SERVLET_CONTEXT_PATH=/example
$ gradle bootRun 
```
